title: Add Printer

:::info Who this is for
Manager or authorized staff setting up devices.
:::

## Before you start

- You have printer IP and port
- Printer is powered on and connected to same network

## Steps

1. Open `Devices`.
2. Select `Printer` tab.
3. Tap `Add Printer`.
4. Fill `Name`, `Printer Type`, `Connection`, `IP Address`, and `Port`.
5. Tap `Save`.
6. Tap `Test Print`.

## What you should see

- Printer appears in list
- Status shows connected/ready
- Test print success dialog appears

## If something goes wrong

:::warning Cannot print
Check IP/port, paper, and network, then retry `Test Print`.
:::

## When to ask manager/admin

- `POS Code` is required but you cannot edit settings
- Printer remains offline after local checks
