:::info Who this is for
All front-line staff.
:::

## Before you start

- Terminal is online
- You have your PIN

## Steps

1. (Optional) Tap the language quick-switch button (`EN` / `繁` / `简`) if you need to change login language.
2. On login screen, tap PIN numbers.
3. Confirm PIN is entered correctly.
4. Tap `SIGN IN`.

## What you should see

- You move to `Ordering` page after successful login.

![Login PIN screen](/img/manual/en/getting-started/gs-06-login-passcode-screen.png)

## If something goes wrong

:::warning Login failed
Retry once after checking PIN carefully.
:::

## When to ask manager/admin

- PIN still fails after retry
- Subscription/offline warning blocks login
