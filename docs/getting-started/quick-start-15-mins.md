Use this page for onboarding a new cashier quickly on one terminal.

## 0 to 3 mins: Terminal Ready

1. Power on device and confirm internet is stable.
2. Complete initialization if setup wizard appears.
3. If prompted, complete first order-data sync.

Reference: `getting-started/first-time-initialization`

## 3 to 6 mins: Sign In and Device Check

1. Enter PIN and tap `SIGN IN`.
2. If required, tap roster button and `Clock In`.
3. Open `Devices` and confirm printer status is healthy.
4. Run one `Test Print`.

Reference: `getting-started/access-and-navigation`, `client/start-shift`

## 6 to 10 mins: Take First Order

1. Open `Ordering`.
2. Select `Dine-In` or `Takeaway`, then select table/order.
3. Add item(s), adjust quantity if needed.
4. Use function menu only when required.
5. Tap `Send` or go straight to `Checkout`.

Reference: `client/sales-flow`

## 10 to 13 mins: Collect Payment

1. In checkout, stay on `Amount` for normal payment.
2. Select payment method.
3. Enter amount and tap `Pay`.
4. Confirm transaction is completed.

Reference: `client/checkout-and-payments`

## 13 to 15 mins: Basic Recovery and Escalation

1. If payment fails, retry once only after checking status.
2. Do not force close app during gateway processing.
3. Escalate to manager for void/return/day-end or settings changes.

Reference: `client/payment-errors-and-retry`, `getting-started/roles-and-permissions`

## Ready-to-Operate Checklist

- Login works with your PIN
- Printer test print is successful
- You can open table/order screen
- You can complete one checkout flow
