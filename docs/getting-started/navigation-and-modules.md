## Left Sidebar Modules

Staff terminals use these sidebar modules:

- `Ordering`
- `Online Orders`
- `Orders Overview`
- `Inventory`
- `Devices`
- `Reports`
- `More Tools`
- `Settings`
- `Info`

![Left sidebar modules](/img/manual/en/getting-started/gs-08-left-sidebar-modules.png)

## Main Operator Areas

- `Ordering`: table selection and order entry
- `Order` page: cart, qty changes, send/checkout
- `Checkout`: payment collection and split/deposit modes
- `Completed order` panel: return/void/reprint/modify payment
- `More Tools`: manager actions such as day-end and pay in/out

## Table Screen Navigation

Table page supports:

- Dine-in/Takeaway mode tabs
- Section tabs (if multiple sections)
- `Split Table` action for dine-in mode
- `Auto Open` action when auto-open mode is enabled

Footer counters show:

- `Available`
- `Occupied`
