:::info[Who this is for]
Managers and authorized staff performing operational controls.
:::

## Before you start

- You are signed in with required role rights
- Store policy allows manager-tool actions on this terminal

## What this screen is for

`More Tools` is the manager operations hub.

:::note[Scope boundary]
`More Tools` is for manager operational controls (`Day-end`, `Pay In / Pay Out`, drawer/queue tools).
For hardware/device setup and printer troubleshooting, use `Devices`.
:::

## Available tools

- `Open Drawer`
- `Day-end`
- `Pay In / Pay Out`
- `Reset Table Counters` (quick-order mode only)
- `Pickup Controller` (if enabled)
- `Queue Controller` (if enabled)

## Steps

1. Tap `More Tools`.
2. Select the required tool card.
3. Complete the confirmation flow for that tool.

## If something goes wrong

:::warning[Card is missing]
Some tools are role/platform/configuration dependent.
:::

## When to ask manager/admin

- Tool access is blocked by permissions
- Quick-order/pickup/queue controls do not match store setup
