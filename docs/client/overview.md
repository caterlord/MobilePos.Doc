## What the Client App Covers

The Client app handles store-side operations from login to day-end.

## Core Workflow Blocks

1. `Start Shift`: login, roster, device checks
2. `Sales Flow`: table/order, items, modifiers, send/close
3. `Checkout and Payments`: amount/split/deposit/split-item payment
4. `Refunds and Voids`: in-progress void and completed-order return flows
5. `End of Day`: manager day-end and settlement controls
