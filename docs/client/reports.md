title: Reports

:::info[Who this is for]
Managers and owners reviewing store performance and audit records.
:::

## Before you start

- You are signed in with report access
- Printer is ready if hardcopy is needed

## What this screen is for

Use `Reports` to select a report type, choose date/context, view results, and print when needed.

## Steps

1. Tap `Reports`.
2. Select the report type from the report list.
3. Choose date and any required options (for example drawer code, if prompted).
4. Wait for report to load.
5. Tap `Print` if a print copy is needed.

## What you should see

- Report selector, date controls, and `Print` button
- Report content panel updates based on selected report

## If something goes wrong

:::warning[Report is empty or slow]
Confirm date/range first, then retry once.
:::

## When to ask manager/admin

- Required report type is missing
- Printed report does not match on-screen result
