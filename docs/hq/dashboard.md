# Dashboard Overview

:::info Who this is for
Store owners and general managers monitoring top-level daily activity and active connected stores.
:::

## Live Dashboard

The Dashboard provides a real-time overview of your connected shops and fundamental daily metrics. 

![HQ Dashboard Overview](/img/manual/en/hq/dashboard.png)

## Available Metrics

On the main screen, you can see:
- **Active Locations**: Which of your stores or branches are currently online.
- **Top Level Summaries**: Quick glimpses into today's activity without needing to run a full report.

## How to navigate HQ
Use the left sidebar to jump into specific configuration areas such as:
- **Organization Management**: Adding shops and handling billing.
- **Store Settings**: Adjusting store opening hours, parameters, and floor plans.
- **POS Settings**: Toggling payment methods and cashier permissions.
- **Menus**: Creating items and adjusting pricing.

To update your own personal login details or switch between different companies/organizations, click your profile avatar in the navigation bar.
