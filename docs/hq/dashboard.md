:::info[Who this is for]
Store owners and general managers monitoring top-level daily activity and active connected stores.
:::

## Before you start

- Sign in to the correct organization or brand.
- Decide whether you are checking status only or trying to find the next setup page.

## Open this page

Open `Dashboard`.

## What this page is for

`Dashboard` is the main landing page after sign-in. Use it to check high-level activity and jump to the next area of HQ.

![HQ Dashboard Overview](/img/manual/en/hq/dashboard.png)

## Main sections

- **Active Locations**: which stores are currently online or active.
- **Top-level summaries**: quick daily or current-period indicators.
- **Navigation shortcuts**: links to the main HQ setup areas.

## How to use this page

Use the dashboard to decide which section you need next:

- go to [Organization Management](./organization.md) for HQ members and business-level setup
- go to [Store Settings Overview](./store-settings/overview.md) for shop-specific setup
- go to [POS Settings Overview](./pos-settings/overview.md) for POS behavior and payment setup
- go to [Menu & Catalog Overview](./menu-catalog/overview.md) for item and menu maintenance
- go to [Reports Overview](./reports/overview.md) for analytics

To update your own personal login details or switch between different companies/organizations, click your profile avatar in the navigation bar.

## What changes after you save

The dashboard is mainly a monitoring and navigation page. Use it to locate the correct setup section before you edit data elsewhere.

## If something goes wrong

- Confirm you are in the correct organization or brand.
- If the numbers look wrong, open the related report or setup section before editing anything.

## When to ask owner/admin

- You cannot find the correct brand or organization.
- The dashboard shows unexpected store status or metrics and you do not know which section owns the issue.
