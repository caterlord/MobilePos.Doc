title: Terminology
---

| Term / Label | Meaning |
| --- | --- |
| Ordering | Main sales module for table/order operations |
| Table | Floor/table selection page before entering an order |
| Order | Transaction detail page with cart and actions |
| Checkout | Payment panel for amount, split, deposit, or split-by-item |
| Void Txn | Cancel an in-progress transaction |
| Return | Create a negative transaction from a completed transaction |
| Reopen | Reopen a checked/completed transaction for further action |
| Pay In / Pay Out | Cash drawer in/out recording action |
| Day-end | Business-day closing process from Manager Tools |
| POS Code | Terminal code required for many device/config operations |
