## 收銀端會涵蓋什麼

收銀端會處理門店端操作，由登入一直到日結。

## 核心流程區塊

1. `開班`：登入、出勤、裝置檢查
2. `銷售流程`：堂食 / 訂單、品項、加配料、傳送 / 結束
3. `結賬與付款`：金額 / 分單 / 訂金 / 按品項分單付款
4. `退款與作廢`：進行中作廢和已完成訂單退回流程
5. `日結`：經理日結和結算控制

## 主要畫面

- `登入`
- `點餐`
- `線上訂單`
- `訂單總覽`
- `庫存`
- `訂單`（未完成 / 已完成）
- `結賬`
- `裝置`
- `報表`
- `更多工具`
- `設定`
- `資訊`

## 如何使用本手冊

- 需要逐班處理：**當班作業**
