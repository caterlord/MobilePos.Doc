---
title: 整合總覽
---

:::info[適用對象]
負責將 X1 HQ 連接到外部夥伴，並維護這些夥伴連線的管理員。
:::

## 開始前

- 確認正確的品牌。
- 確認夥伴是否已透過 `Marketplace` 訂閱或安裝。
- 在編輯前先準備好識別碼、Token 或夥伴提供的設定資料。

## 本節用途

使用 `Integrations` 來管理 X1 與第三方服務之間的夥伴連線。

本節與店舖硬體設定不同：

- 使用 `Integrations` 來處理夥伴連線與夥伴店舖設定
- 使用 `Store Settings` -> `Device Settings` 來處理本地裝置與終端機設定

## 如何使用本節

- [Partner 設定](./partner-settings.md)：檢視已安裝的夥伴、新增夥伴，並依店舖管理夥伴設定。

如果您還需要先啟用可選服務，請使用 [市集](../marketplace/catalog.md)。

## 常見工作

- 訂閱後新增夥伴：先從 [Partner 設定](./partner-settings.md) 開始。
- 檢查夥伴是否已設定到正確店舖：使用 [Partner 設定](./partner-settings.md)。
- 確認夥伴是否需要先訂閱市集：檢查 [市集](../marketplace/catalog.md)。

## 儲存後的影響

整合設定可能需要多個步驟：

- 市集訂閱
- 夥伴安裝
- 店舖專屬設定

請分開儲存並逐層驗證。

## 如發生問題

- 確認夥伴是否已安裝或已訂閱。
- 確認正確的品牌與店舖。
- 檢查問題是出在夥伴設定，還是其他區塊，例如 `Payment Vendors` 或 `Device Settings`。

## 何時要請店主／管理員協助

- 您沒有夥伴的憑證。
- 整合會影響財務、網上點餐或正式結帳。
- 夥伴已安裝，但資料沒有如預期流動。
