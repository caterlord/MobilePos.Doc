:::info[適用對象]
需要在不等待前台發版的情況下覆寫顧客端網上點餐文案的管理員。
:::

## 開始前

- 確認已選擇正確的品牌。
- 先決定要修改哪個語言與哪個前台渠道。
- 確認您要做的是介面文案覆寫，而不是基礎菜單或項目名稱翻譯。

## 開啟此頁面

開啟 `Online Ordering` -> `UI i18n`。

## 本頁用途

`UI i18n` 可讓您覆寫網上點餐應用程式中的顧客前台文案。

您可以在這裡：

- 選擇要修改的語言
- 指定某個訂單渠道
- 搜尋要修改的文案 key
- 儲存自訂覆寫文字
- 刪除覆寫並恢復系統預設文案

本頁只會改變前台介面文字，例如按鈕、標籤與提示，不會取代其他地方維護的菜單項目翻譯。

![UI 翻譯](/img/hq/online-ordering/oo_translations_page.png)

## 主要區塊

### 語言選擇

選擇要修改的語言，例如 `Traditional Chinese (HK)`。

### 訂單渠道選擇

選擇要套用覆寫的前台或訂單渠道，例如 `WEB`。

### Key 搜尋

使用搜尋列找出要修改的文案 key。

如果您不知道 key 名稱，可以先搜尋顧客實際看到的字詞，例如 `Dine-In`。

### 覆寫值

在覆寫欄位輸入新的文字，然後儲存。

如果您想刪除自訂覆寫並恢復系統預設文案，請先把覆寫欄位完全清空，再按儲存。

## 步驟

1. 開啟 `Online Ordering` -> `UI i18n`。
2. 選擇目標語言。
3. 選擇目標訂單渠道。
4. 搜尋要修改的文字 key。
5. 輸入新的覆寫值。
6. 選擇 `Save`。

## 儲存後的影響

目標前台會在指定語言與訂單渠道中使用新的文字。

這類變更只會更新顧客前台介面文案，不需要重新部署前台程式。

## 如何檢查你的變更

1. 重新打開相同 key，確認覆寫值仍然存在。
2. 以相同語言與渠道打開目標前台。
3. 確認畫面上對應位置已顯示新文案。

## 如發生問題

- 確認您修改的是正確語言與訂單渠道。
- 確認您修改的是正確 key，而不是另一個相似標籤。
- 如果您想刪除覆寫，請先把欄位完全清空再儲存。
- 如果前台仍顯示舊文字，請重新整理目標流程並再確認渠道選擇。

## 何時要請店主／管理員協助

- 文案變更會影響營業中的顧客流程。
- 您不確定這應該是本地覆寫，還是永久產品預設。
- 文案修改涉及法規、付款或合規風險。
