import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

:::info[規則]
使用 `Void` 處理進行中的交易。使用 `Return` 處理已完成的交易。
:::

## 步驟

<Tabs>
  <TabItem value="void" label="作廢">
1. 開啟進行中的訂單。
2. 按功能選單。
3. 在 `Select Function` 中按 `Void Txn`。
4. 如果 `Void Txn` 在其他頁，按 `Previous` 或 `Next`。
5. 確認提示。
  </TabItem>
  <TabItem value="return" label="退回">
1. 開啟已完成交易。
2. 按 `Return`。
3. 確認提示。
4. 完成退款付款流程。
  </TabItem>
</Tabs>

## 如果出現問題

:::warning[操作被阻止]
先清除尚未提交的修改，再重試一次。
:::

## 何時需要找經理／管理員

- 高金額退款 / 作廢
- 第三方連結付款有修改限制
