---
title: 分拆或合併交易
---

:::info[適用對象]
需要在付款前把品項在交易之間移動的收銀員和更班經理。
:::

## 開始前

- 您已登入並位於訂單畫面。
- 您可以打開功能選單（三點按鈕）。
- 您已按 `Send` 送出尚未提交的品項修改。
- 您已了解兩者的分別：
  `Split / Merge Txn` 是在結賬前把品項在交易之間移動。
  `Split By Item` 是結賬時的付款分拆流程。

## 步驟

1. 按功能選單（三點按鈕）。
2. 在 `Select Function` 中按 `Split / Merge Txn`。
3. 如果 `Split / Merge Txn` 在其他頁，按 `Previous` 或 `Next`。
4. 在 `Select Table` 中按目標桌台。
5. 在來源一側按要選擇的品項。
6. 按 `>` 移動已選品項，或按 `All` 移動全部品項。
7. 按 `Confirm`。
8. 開啟目標交易，確認已移動的品項。

## 您會看到

- `Split / Merge Txn` 視窗會顯示來源和目標兩個區塊。
- 在您確認前，已移動的品項會先出現在目標一側。
- 按 `Confirm` 後，新交易狀態會儲存。

## 如果出現問題

- 如果看到 `Have unsend item(s), cannot split Txn`，請先按 `Send` 再重試。
- 如果看到 `Please transfer the item`，請先移動至少一個品項再按 `Confirm`。
- 如果看到 `Table is placing order`，請改選另一張目標桌台。
- 如果看到 `Unable to split order`，請先重試一次。

## 何時需要找經理／管理員

- `Manager only`：如門店政策要求，跨桌台合併 / 分拆需先獲批准。收銀員可先保留目前交易並繼續服務。
- 如果您不確定應該使用 `Split / Merge Txn` 還是結賬時的 `Split By Item`，請找經理。
- 如果在有效桌台上重試後仍然失敗，請找管理員。
