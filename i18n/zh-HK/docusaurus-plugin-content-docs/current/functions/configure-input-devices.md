---
title: 設定輸入裝置
---

:::info[適用對象]
設定掃描器 / 輸入硬件的經理或授權員工。
:::

## 開始前

- 您已登入並具備裝置設定權限
- 已設定 `POS Code`

## 步驟

1. 開啟 `Devices`。
2. 開啟 `Input Devices` 分頁。
3. 如有需要，按 `Info` 檢查狀態。
4. 按目標裝置的編輯圖示。
5. 更新所需欄位，然後按 `Save`。

## 您會看到

- 輸入裝置清單和狀態指示
- 每列都有 `Info` 和編輯操作

> Screenshot pending: `client/cl-35-devices-input-devices-tab-list.png`

對於條碼 / QR 掃描器，編輯視窗可能包括：

- `掃描輸入來源`
- `相機`
- `USB / 藍牙`
- `掃描預覽`
- `相機`（鏡頭方向選擇）
- `測試 USB/藍牙掃描器`
- `啟用相機備用`

> Screenshot pending: `functions/fn-input-barcode-scanner-dialog.png`

使用 USB 或藍牙掃描器時，選擇 `USB / 藍牙` 作為預設掃描方式。先在 `測試 USB/藍牙掃描器` 掃描，確認掃描值出現後再按 `Save`。

如果希望 POS 優先開啟相機，請選擇 `相機`。如果預設使用 USB/藍牙，但員工可能需要用相機備用，請開啟 `啟用相機備用`。

## 如果出現問題

:::warning[掃描器無法讀取]
請先確認所選掃描輸入來源。USB/藍牙掃描器需要能像鍵盤一樣輸入文字並傳送 Enter。相機掃描請檢查相機權限和鏡頭方向設定，然後重試。
:::

## 何時需要找經理／管理員

- 輸入裝置儲存後仍然不可用
- 所需的掃描器選項不存在
