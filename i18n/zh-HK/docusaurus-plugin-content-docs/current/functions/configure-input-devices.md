---
title: 設定輸入裝置
---

:::info[適用對象]
設定掃描器 / 輸入硬件的經理或授權員工。
:::

## 開始前

- 您已登入並具備裝置設定權限
- 已設定 `POS Code`

## 步驟

1. 開啟 `Devices`。
2. 開啟 `Input Devices` 分頁。
3. 如有需要，按 `Info` 檢查狀態。
4. 按目標裝置的編輯圖示。
5. 更新所需欄位，然後按 `Save`。

## 您會看到

- 輸入裝置清單和狀態指示
- 每列都有 `Info` 和編輯操作

> Screenshot pending: `client/cl-35-devices-input-devices-tab-list.png`

對於條碼 / QR 掃描器，編輯視窗可能包括：

- `Connection`
- `IP Address` / `Port`（IP 模式）
- `Scanner Preview`
- `Camera`（鏡頭方向選擇）

> Screenshot pending: `functions/fn-input-barcode-scanner-dialog.png`

## 如果出現問題

:::warning[掃描器無法讀取]
請先確認所選連線模式，以及 camera / host 網絡設定，然後重試。
:::

## 何時需要找經理／管理員

- 輸入裝置儲存後仍然不可用
- 所需的掃描器選項不存在
