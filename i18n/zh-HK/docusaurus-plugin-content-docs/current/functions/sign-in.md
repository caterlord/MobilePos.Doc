:::info[適用對象]
所有前線員工。
:::

## 開始前

- 終端已連線
- 您已備妥 PIN 碼

## 步驟

1. 如需變更登入語言，可先按語言快速切換按鈕（`EN` / `繁` / `简`）。
2. 在登入畫面輸入 PIN 碼。
3. 確認 PIN 已正確輸入。
4. 按 `SIGN IN`。

## 您會看到

- 登入成功後，畫面會切換到 `Ordering`。

![Login PIN screen](/img/manual/en/getting-started/gs-06-login-passcode-screen.png)

## 如果出現問題

:::warning[登入失敗]
先仔細確認 PIN 碼，再重試一次。
:::

## 何時需要找經理／管理員

- 重試後仍然無法登入
- 訂閱或離線提示阻止登入
