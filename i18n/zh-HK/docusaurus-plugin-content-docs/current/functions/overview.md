:::info[本節用途]
Use-case 頁面 最適合訓練。Function 頁面 最適合您需要快速完成一個精確操作的時候。
:::

當您想要每頁只看一個工作時，請使用本節，例如：

- Ordering functions（折扣、服務費、變更桌台、外賣訂單聯絡資料、訂單類型切換、分拆 / 合併交易、會員登入）
- Checkout 及已完成訂單 functions（收款、作廢 / 退回、重開、重印）
- 裝置 functions（印表機管理、付款 / 輸入 / X1 設定）
- 更多工具 functions（代收 / 代支、日結）

如需完整索引，請參考 [Function and Feature Map](../reference/function-feature-map)。
