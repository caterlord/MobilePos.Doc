本手冊供 **POS X1 Client 應用程式** 的店舖團隊使用。

本文件根據已確認的產品行為與目前 UI 標籤編寫。
內容重點是收銀員與經理在 Client 應用程式的日常操作。

## 範圍

- 日常登入與模組導覽
- 桌台／訂單作業
- 結帳與付款處理
- 作廢、退回與重印流程
- 經理工具與日結
- 裝置健康檢查

## 來源依據

- 最新已發佈的產品行為
- 產品中已確認的 UI 標籤
- 本儲存庫中已驗證的文件更新

## 閱讀方式

1. 新店舖設定：**Getting Started**
2. 日常操作：**Client (POS App)**
3. 問題處理：**參考**

## 內容組織方式

本手冊目前分為三個並行視角：

- **依使用情境**：以步驟方式呈現操作指南，例如開班、銷售、結帳、退貨、日結
- **依角色**：說明收銀員與經理的權限與升級處理界線
- **依功能**：每個功能一頁，方便快速查找，並附完整功能索引

使用 **Function Guides** 來查看單一任務頁面，並使用 **參考 -> Function & Feature Map** 來查看完整索引。
