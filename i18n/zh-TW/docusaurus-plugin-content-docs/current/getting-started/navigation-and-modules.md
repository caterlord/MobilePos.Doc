## 左側欄模組

店員終端機使用以下側欄模組：

- `Ordering`
- `Online Orders`
- `Orders Overview`
- `Inventory`
- `Devices`
- `Reports`
- `More Tools`
- `Settings`
- `Info`

![左側欄模組](/img/manual/en/getting-started/gs-08-left-sidebar-modules.png)

## 主要操作區域

- `Ordering`：桌台選擇與訂單輸入
- `Order` 頁面：購物車、數量變更、傳送／結帳
- `Checkout`：收款與分拆／訂金模式
- `Completed order` 面板：退回／作廢／重印／修改付款
- `More Tools`：經理操作，例如日結與現金收付

## 桌台畫面導覽

桌台頁面支援：

- `Dine-In` / `Takeaway` 模式分頁
- 區塊分頁（若有多個區塊）
- `Split Table` 功能（堂食模式）
- `Auto Open` 功能（若已啟用自動開單）

頁尾會顯示：

- `Available`
- `Occupied`
