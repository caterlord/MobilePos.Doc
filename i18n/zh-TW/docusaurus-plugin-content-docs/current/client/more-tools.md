:::info[適用對象]
經理與授權員工，用來執行營運控制。
:::

## 開始前

- 您已登入，且具備需要的角色權限
- 店舖政策允許這台收銀機執行經理工具

## 此畫面用途

`More Tools` 是經理營運控制中心。

:::note[範圍界線]
`More Tools` 用來處理經理營運控制，例如 `Day-end`、`Pay In / Pay Out`、抽屜／佇列工具。
若要做硬體／裝置設定或印表機疑難排解，請使用 `裝置`。
:::

## 可用工具

- `Open Drawer`
- `Day-end`
- `Pay In / Pay Out`
- `Reset Table Counters`（僅限快速點餐模式）
- `Pickup Controller`（如果已啟用）
- `Queue Controller`（如果已啟用）

## 步驟

1. 點選 `More Tools`。
2. 選擇需要的工具卡片。
3. 完成該工具的確認流程。

## 如發生問題

:::warning[工具卡片不見了]
有些工具會依角色、平台或設定而顯示不同。
:::

## 何時要找經理／管理員

- 權限阻止您使用該工具
- 快速點餐、取餐或佇列控制與店舖設定不一致
