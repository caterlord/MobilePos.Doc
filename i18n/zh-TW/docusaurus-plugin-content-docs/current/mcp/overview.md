# Model Context Protocol (MCP)

使用 X1 的 MCP 伺服器，讓 AI 代理可與 X1 API 互動。

X1 的 Model Context Protocol (MCP) 伺服器提供一組工具，讓 AI 代理可安全地查詢 X1 API、手冊頁面及支援文章。

## 如何開始

1. 如要連接 Cursor、Claude Desktop 或 ChatGPT，請按英文頁面的最新設定步驟操作。
2. 如要確認支援的工具清單，請參考下方的 MCP 工具說明。
3. 如要提供 MCP 相關意見，請使用 `mcp@x1.tech`。

:::info[語言說明]
MCP 設定檔、OpenAPI URL 與工具名稱會保留原文，以免影響實際接入設定。
:::
