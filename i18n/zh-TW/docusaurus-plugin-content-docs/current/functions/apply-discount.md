:::info[適用對象]
需要在結帳前調整訂單總額的收銀員。
:::

## 開始前

- 您已登入，且目前在 `Ordering`
- 您已開啟正確的訂單
- 訂單中至少有一個品項

## 步驟

<Tabs>
  <TabItem value="apply-discount" label="套用折扣">
1. 在 `Ordering` 開啟目標訂單。
2. 點選功能按鈕。
3. 在 `選擇功能` 中點選 `套用折扣`。
4. 如果 `套用折扣` 在其他頁面，請點選 `上一頁` 或 `下一頁`。
5. 從清單中選擇折扣。
6. 確認折扣。
  </TabItem>
  <TabItem value="cancel-discount" label="取消折扣">
1. 在 `Ordering` 開啟目標訂單。
2. 點選功能按鈕。
3. 在 `選擇功能` 中點選 `取消折扣`。
4. 如果 `取消折扣` 在其他頁面，請點選 `上一頁` 或 `下一頁`。
5. 選擇您要移除的折扣。
6. 確認取消。
  </TabItem>
</Tabs>

## 你應該看到什麼

- 套用折扣後，訂單會出現折扣行，總額也會變低
- 取消折扣後，折扣行會移除，總額會恢復

![套用 Discount dialog: 選擇 a 折扣 供 這 訂單](/img/manual/en/client/cl-09-apply-discount-dialog.png)

## 如發生問題

- 如果 `套用折扣` 或 `取消折扣` 不見了，請檢查該帳號的角色權限
- 如果折扣清單是空的，請檢查店舖折扣設定並重新開啟訂單
- 如果確認後總額沒有更新，請關閉訂單再重新開啟檢查一次

## 何時要找經理／管理員

- 出現 `Manager only` 的授權提示，且您無法繼續
- 折扣選項與店舖政策不符
- 已套用折扣，但結帳仍被付款規則警告阻擋
