---
title: 設定輸入設備
---

:::info[適用對象]
負責設定掃描器或輸入硬體的經理或授權員工。
:::

## 開始前

- 您已登入，且具備裝置設定權限
- `收銀機代碼` 已設定

## 步驟

1. 開啟 `裝置`。
2. 開啟 `Input Devices` 分頁。
3. 如果需要，點選 `Info` 檢查狀態。
4. 點選目標設備上的編輯圖示。
5. 更新需要的欄位，然後點選 `儲存`。

## 你應該看到什麼

- 輸入設備清單與狀態指示器
- 每一列都有 `Info` 與編輯動作

> 截圖待補：`client/cl-35-devices-input-devices-tab-list.png`

對於條碼／QR code 掃描器，編輯對話框可能包含：

- `Connection`
- `IP Address` / `Port`（IP 模式時）
- `Scanner Preview`
- `Camera`（相機方向選擇）

> 截圖待補：`functions/fn-input-barcode-scanner-dialog.png`

## 如發生問題

:::warning[掃描器無法讀取]
先確認所選連線模式，以及相機／主機網路設定，然後重試。
:::

## 何時要找經理／管理員

- 儲存後輸入設備仍然不可用
- 需要的掃描器選項不見了
