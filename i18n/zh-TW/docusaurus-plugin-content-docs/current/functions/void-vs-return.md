import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

:::info[規則]
進行中的交易使用 `取消交易`。已完成的交易使用 `退回`。
:::

## 步驟

<Tabs>
  <TabItem value="void" label="作廢">
1. 開啟進行中的訂單。
2. 點選功能選單。
3. 在 `選擇功能` 中點選 `取消交易`。
4. 如果 `取消交易` 在其他頁面，請點選 `上一頁` 或 `下一頁`。
5. 確認提示。
  </TabItem>
  <TabItem value="return" label="退回">
1. 開啟已完成交易。
2. 點選 `退回`。
3. 確認提示。
4. 完成退款付款流程。
  </TabItem>
</Tabs>

## 如發生問題

:::warning[動作被阻止]
先清除待處理編輯，然後重試一次。
:::

## 何時要找經理／管理員

- 高金額退款或作廢
- 第三方連結付款限制
