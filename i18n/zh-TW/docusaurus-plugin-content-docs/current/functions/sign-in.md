:::info[適用對象]
所有第一線員工。
:::

## 開始前

- 收銀機已連線
- 您有自己的 PIN

## 步驟

1. （選用）如果您需要切換登入語言，先點選語言快速切換按鈕（`EN` / `繁` / `简`）。
2. 在登入畫面輸入 PIN 數字。
3. 確認 PIN 已輸入正確。
4. 點選 `登入`。

## 你應該看到什麼

- 成功登入後，您會進入 `Ordering` 頁面。

![登入 PIN 畫面](/img/manual/en/getting-started/gs-06-login-passcode-screen.png)

## 如發生問題

:::warning[登入失敗]
先仔細檢查 PIN，然後重試一次。
:::

## 何時要找經理／管理員

- 重試後 PIN 仍然失敗
- 訂閱或離線警告阻止登入
