# 網上菜單

![網上菜單設定](/img/hq/online-ordering/oo_menus_page.png)

**網上菜單** 模組會決定已發佈的分類要如何組合成可瀏覽的數位目錄。

它的運作方式類似 POS 菜單功能，但只適用於網頁與應用程式使用者。

## 菜單組合

「菜單組合」就是外層包裝，例如 `Lunch Delivery Menu` 或 `Dinner Pickup Menu`。

顧客開啟您的店舖連結時，看到的就是某一個特定的菜單組合。

### 建立菜單
1. 點選 **New Menu** 建立主體。
2. 定義 **代碼**（供 API 串接使用）與 **顯示名稱**（在網頁版應用程式上會顯示為頁面標題）。
3. 選擇要映射到這個菜單的 **分類**（也就是前一個步驟中設定好的分類）。
4. 以拖放方式調整分類順序，讓顧客手機上顯示的順序符合您的需求。

### 指定店舖
網上菜單是全域性的，但必須明確指派到實體店舖。
如果 `Shop A` 與 `Shop B` 的菜單內容完全相同，您只需要建立一次菜單，然後指派給兩家店。
但如果 `Shop A` 有賣酒、`Shop B` 沒有，您就應該建立兩套不同的網上菜單，並分別對應。
