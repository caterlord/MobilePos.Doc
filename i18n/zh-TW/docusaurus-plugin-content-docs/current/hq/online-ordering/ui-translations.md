**UI 翻譯** 功能提供接近工程層級的控制，讓您直接調整顯示在顧客端網頁應用程式上的實際文字字串。

與其送出支援單去改按鈕文字，HQ 管理員可以直接從這個頁面覆寫應用程式字典。

## 運作方式

顧客端網頁應用程式是透過標準化的 `key` 建構，例如 `checkout.button.confirm`。
預設情況下，系統會從全域資料庫讀取英文值，例如 `Confirm Order`。

這個頁面讓您可以針對特定語言或特定訂單通路，注入 **覆寫值**。

### 逐步覆寫

1. **選擇語言**：先選擇您要編輯的語系，例如 `Traditional Chinese (HK)`。
2. **選擇訂單通路**：選擇要套用覆寫的前台，例如 `WEB`。
3. **搜尋鍵值**：使用搜尋列找出要修改的文字，例如搜尋 `Dine-In`。
4. **輸入覆寫內容**：在文字框中輸入新的品牌化文案，例如把 `Dine-In` 改成 `Eat Here Today!`。
5. 點選 **儲存**。

顧客端網頁應用程式會立即反映新的文字內容，不需要部署程式碼或更新應用程式。

:::warning[覆寫清理]
如果您想把某個欄位清空，請不要只留一半內容。若要刪除覆寫並回復系統預設，請在儲存前把文字框完全清空。
:::
