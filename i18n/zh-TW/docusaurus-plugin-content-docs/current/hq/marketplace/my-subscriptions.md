title: 我的訂閱

:::info[適用對象]
需要檢查 X1 POS 裝置席位，或管理目前品牌啟用中市集訂閱的管理員。
:::

## 開始前

- 在左上角選擇器確認正確的工作區和品牌。
- 只有在目前平板已損壞、遺失或永久替換時，才釋放 POS 裝置席位。
- 釋放舊裝置前，先確認替換平板已準備好註冊。
- 先確認移除市集訂閱是否會影響營業中的店鋪或整合。

## 開啟此頁面

開啟 `我的訂閱`。

此頁有兩個分頁：

- `X1 產品`：檢查 X1 POS 裝置席位，並從舊平板釋放席位。
- `市集`：檢查和管理市集產品訂閱。

## 本頁控制內容

`我的訂閱` 控制兩類訂閱。

### X1 POS 裝置席位

使用 `X1 產品` 檢查所選品牌的 POS 訂閱席位。

`X1 POS 裝置席位` 表格會顯示：

| 欄位 | 說明 |
| --- | --- |
| `店鋪` | 席位所屬店鋪。 |
| `狀態` | `已佔用` 表示已有裝置使用席位。`可用` 表示替換平板可註冊到此席位。 |
| `裝置` | 可用時顯示 POS 裝置名稱。如果未提供裝置名稱，HQ 會顯示縮短的裝置 ID 和 `未提供裝置名稱`。 |
| `啟用時間` | 裝置佔用席位的時間。 |
| `最後上線` | 該 POS 裝置最後一次檢查訂閱的時間。 |
| `訂閱期間` | 如有設定開始或結束日期，會顯示席位期間。 |
| `操作` | 已佔用席位的 `釋放裝置` 動作。 |

表格上方摘要會顯示 `總席位`、`已使用席位` 和 `可用席位`。

### 市集訂閱

使用 `市集` 檢查目前品牌已訂閱的市集產品。

依產品不同，頁面可能會提供以下動作：

- `設定`
- `取消訂閱`

`設定` 只會出現在有直接設定頁面的產品。由市集管理的 `3rd Party Platforms` 通常要改到 [General Settings](../online-ordering/general-settings.md) 與 [3rd Party Platforms](../online-ordering/third-party-platforms.md) 繼續處理。

## 步驟

### 釋放已替換的 POS 平板

1. 開啟 `我的訂閱`。
2. 選擇 `X1 產品`。
3. 檢查 `總席位`、`已使用席位` 和 `可用席位`。
4. 找到受影響店鋪和裝置的列。
5. 檢查 `裝置`、`啟用時間` 和 `最後上線`，確認這是舊平板。
6. 在 `操作` 選擇 `釋放裝置` 圖示。
7. 閱讀警告訊息。
8. 確認釋放。

確認後，該列會變為可用，替換平板可以註冊到釋放出的席位。

### 管理市集訂閱

1. 開啟 `我的訂閱`。
2. 選擇 `市集`。
3. 找到要管理的訂閱。
4. 如果產品有設定頁，選擇 `設定`。
5. 如果產品已不再需要，選擇 `取消訂閱`。
6. 如系統有提示，請確認取消訂閱。

## 儲存後的影響

釋放 POS 裝置席位不會取消訂閱。它只會清除舊裝置佔用，讓另一部平板可以註冊。舊平板再次上線並檢查訂閱時，會失去存取權。

POS app 版本如會傳送裝置名稱，裝置啟用或驗證訂閱後，HQ 便會顯示裝置名稱。較早已佔用的席位，在該平板使用更新後 app 回報前，可能會顯示縮短裝置 ID 和 `未提供裝置名稱`。

移除市集訂閱後，相關功能或整合可能會立即不可用。

對於由市集管理的線上點餐平台，取消訂閱也可能移除相關渠道可用性，並清除店鋪層級已儲存的平台設定。請將 `取消訂閱` 視為高影響變更。

## 如何檢查你的變更

1. 如已釋放 POS 平板，確認席位由 `已佔用` 變為 `可用`。
2. 註冊替換平板，確認席位重新變為 `已佔用`。
3. 確認替換平板可通過 POS 訂閱檢查。
4. 如保留市集訂閱，確認產品仍出現在清單中。
5. 如移除市集訂閱，確認它已從啟用中訂閱清單消失。
6. 如果您移除了平台訂閱，營業前請重新檢查 [Channel Settings](../online-ordering/channel-mapping.md) 或 [3rd Party Platforms](../online-ordering/third-party-platforms.md)。

## 如發生問題

- 如果顯示 `請選擇品牌以查看 X1 POS 訂閱席位。`，請先選擇正確品牌。
- 如果顯示 `此品牌找不到 X1 POS 訂閱席位。`，請確認該品牌已分配 X1 POS 席位。
- 如果顯示 `X1 POS subscription database is unavailable`，請聯絡支援檢查 production 訂閱資料庫連線。
- 如果 `釋放裝置` 無法使用，代表席位已可用，沒有裝置被指派。
- 如果釋放了錯誤裝置，請重新註冊正確平板，或在舊平板被再次使用前聯絡支援。
- 如果是誤取消訂閱，請回到 [市集](./catalog.md) 重新訂閱（如產品仍可使用）。

## 何時要請店主／管理員協助

- 您不確定應釋放哪一部平板。
- 該平板可能仍在營業中的店鋪使用。
- 訂閱會影響營業中的付款、點餐或財務流程。
- 您不確定是否仍有其他團隊或店鋪在使用該市集產品。
