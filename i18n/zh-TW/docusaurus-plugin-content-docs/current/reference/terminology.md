title: 術語
---

| 術語／標籤 | 說明 |
| --- | --- |
| `Ordering` | 供桌台／訂單作業使用的主要銷售模組 |
| `Table` | 進入訂單前的樓面／桌台選擇畫面 |
| `Order` | 包含購物車與操作按鈕的交易明細頁面 |
| `Checkout` | 可進行金額、分拆、訂金或依品項分拆的付款面板 |
| `Void Txn` | 取消進行中的交易 |
| `Return` | 以已完成交易建立負數交易 |
| `Reopen` | 重新開啟已結帳／已完成的交易以便後續操作 |
| `Pay In / Pay Out` | 現金抽屜的收／付記錄動作 |
| `Day-end` | 由 `Manager Tools` 執行的營業日結束流程 |
| `POS Code` | 多數裝置／設定操作所需的終端機代碼 |
| `Item Mapping` (`Main` / `Alt`) | 各語言顯示的品項名稱來源 |
| `Login Quick Switch` | 登入畫面上的語言按鈕，可快速切換語言 |
