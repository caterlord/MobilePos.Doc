:::info[适用对象]
收银员和轮班经理在营业期间记录钱箱收支时使用。

仅限经理：部分门店会根据角色和权限设置限制 `代收 / 代支`。
:::

## 开始前

- 确认你已登录。
- 确认当前有营业日。
- 确认门店已配置代收/代支原因。

## 步骤

1. 点按 `更多工具`。
2. 点按 `代收 / 代支`。
3. 点按 `代收` 或 `代支`。
4. 在 `请选择代收原因` 或 `请选择代支原因` 中选择原因。
5. 在出现 `请输入金额` 时输入金额。
6. 查看 `确认保存此记录？`，然后点按 `确定`。

## 你应该看到什么

- 新记录会显示在 `代收 / 代支` 列表中，并更新 `原因`、`金额`、`用户` 和 `时间`。
- 如果门店启用了小票打印，系统会打印代收/代支小票。

## 如果出现问题

- 如果看到 `当前没有营业时段，请稍后再试。`，请先检查营业日状态。
- 如果看到 `没有可用的代收原因` 或 `没有可用的代支原因`，说明原因设置缺失。
- 如果看到 `金额必须大于 0`，请输入正数金额。
- 如果看到 `无法保存代收 / 代支记录`，请先检查网络后再重试一次，然后升级处理。

## 何时需要找经理／管理员

- 因角色限制，你无法进入 `代收 / 代支`。
- 原因列表缺失或不正确。
- 保存反复失败，或者打印的小票与记录金额不一致。
