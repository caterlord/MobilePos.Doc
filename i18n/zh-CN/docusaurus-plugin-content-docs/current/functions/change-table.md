---
title: 转台
---

:::info[适用对象]
收银员和轮班经理需要将未结订单转到其他餐台时使用。
:::

## 开始前

- 你已登录并位于订单页面。
- 你已打开正确的订单。
- 你可以打开功能菜单（三个点）。
- 你已使用 `发送` 提交所有待发送的项目修改。
- 你确认目标是一个空餐台。

## 步骤

1. 点按功能菜单（三个点）。
2. 在 `选择功能` 中点按 `转台`。
3. 如果 `转台` 在另一页，点按 `上一页` 或 `下一页`。
4. 在 `转台` 中，如果需要在 `餐台列表` 和 `平面图` 之间切换，请点按右下角的切换按钮。
5. 如果需要查看拆台，点按 `显示分台`。
6. 点按目标餐台。
7. 等待餐台选择器关闭。
8. 如果应用返回餐台列表，请从目标餐台重新打开该订单。

## 你应该看到什么

- 订单不再显示在原餐台上。
- 订单会出现在目标餐台上。
- 你可以继续在目标餐台上点餐。
- 当启用平面图模式时，右下角切换按钮会在 `餐台列表` 和 `平面图` 之间切换。
- 显示拆台后，`显示分台` 会变成 `隐藏分台`。

## 如果出现问题

- 如果看到 `No item to change table`，请先添加至少一个项目。
- 如果看到 `Cannot change table` 和 `Please save all operations before changing table`，先点按 `发送`，再重试。
- 如果在 `平面图` 中找不到目标餐台，请切换到 `餐台列表`。
- 如果找不到目标拆台餐台，请点按 `显示分台`。
- 如果看到 `Please select an empty table`，请改选其他餐台。
- 如果看到 `Unable to change table`，请先刷新订单，再重试一次。

## 何时需要找经理／管理员

- `仅限经理`：涉及特殊处理或争议订单时，才可转台。收银员可继续保留当前餐台并照常服务。
- 如果没有可用的目标餐台，请找经理处理。
- 如果重试一次后仍然失败，请找管理员处理。
