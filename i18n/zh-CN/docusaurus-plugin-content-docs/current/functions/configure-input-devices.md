---
title: 配置输入设备
---

:::info[适用对象]
经理或授权员工配置扫描器/输入硬件时使用。
:::

## 开始前

- 你已登录，并拥有设备设置权限。
- `收银机代码` 已设置。

## 步骤

1. 打开 `设备`。
2. 打开 `输入设备` 页签。
3. 如有需要，点按 `信息` 查看状态。
4. 点按目标设备上的编辑图标。
5. 更新所需字段，然后点按 `保存`。

## 你应该看到什么

- 输入设备列表会显示状态指示。
- 每一行都会显示 `信息` 和编辑操作。

> 截图待补充：`client/cl-35-devices-input-devices-tab-list.png`

对于条码/二维码扫描器，编辑对话框可能包含：

- `扫描输入来源`
- `相机`
- `USB / 蓝牙`
- `扫描预览`
- `相机`（选择前置或后置摄像头）
- `测试 USB/蓝牙扫描器`
- `启用相机备用`

> 截图待补充：`functions/fn-input-barcode-scanner-dialog.png`

使用 USB 或蓝牙扫描器时，选择 `USB / 蓝牙` 作为默认扫描方式。先在 `测试 USB/蓝牙扫描器` 中扫描，确认扫描值出现后再点按 `保存`。

如果希望 POS 优先打开相机，请选择 `相机`。如果默认使用 USB/蓝牙，但员工可能需要用相机备用，请打开 `启用相机备用`。

## 如果出现问题

:::warning[扫描器无法读取]
请先确认所选扫描输入来源。USB/蓝牙扫描器需要能像键盘一样输入文字并发送 Enter。相机扫描请检查相机权限和相机方向设置，然后再重试。
:::

## 何时需要找经理／管理员

- 保存后输入设备仍然不可用
- 缺少所需的扫描器选项
