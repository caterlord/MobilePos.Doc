:::info[适用对象]
所有一线员工。
:::

## 开始前

- 终端机已联网
- 你已准备好 PIN

## 步骤

1. （可选）如果你需要切换登录语言，点按语言快速切换按钮（`EN` / `繁` / `简`）。
2. 在登录页面，点按 PIN 数字键。
3. 确认 PIN 输入正确。
4. 点按 `登录`。

## 你应该看到什么

- 成功登录后，你会进入 `点餐` 页面。

![登录 PIN 页面](/img/manual/en/getting-started/gs-06-login-passcode-screen.png)

## 如果出现问题

:::warning[登录失败]
先检查 PIN，再重试一次。
:::

## 何时需要找经理／管理员

- 重试后 PIN 仍然失败
- 订阅/离线警告阻止登录
