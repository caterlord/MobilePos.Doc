:::info[适用对象]
收银员在结账前需要调整订单总额时使用。
:::

## 开始前

- 你已登录并位于 `点餐`。
- 你已打开正确的订单。
- 订单至少已有一个项目。

## 步骤

<Tabs>
  <TabItem value="apply-discount" label="应用折扣">
1. 在 `点餐` 中打开目标订单。
2. 点按功能按钮。
3. 在 `选择功能` 中点按 `应用折扣`。
4. 如果 `应用折扣` 在另一页，点按 `上一页` 或 `下一页`。
5. 从列表中选择折扣。
6. 确认折扣。
  </TabItem>
  <TabItem value="cancel-discount" label="取消折扣">
1. 在 `点餐` 中打开目标订单。
2. 点按功能按钮。
3. 在 `选择功能` 中点按 `取消折扣`。
4. 如果 `取消折扣` 在另一页，点按 `上一页` 或 `下一页`。
5. 选择你要移除的折扣。
6. 确认取消。
  </TabItem>
</Tabs>

## 你应该看到什么

- 点按 `应用折扣` 后，订单会显示折扣行，且总额会降低。
- 点按 `取消折扣` 后，折扣行会移除，总额会恢复。

![为该订单选择折扣](/img/manual/en/client/cl-09-apply-discount-dialog.png)

## 如果出现问题

- 如果找不到 `应用折扣` 或 `取消折扣`，请检查该账号的角色权限。
- 如果折扣列表为空，请检查门店折扣设置，然后重新打开订单。
- 如果点按确认后总额没有更新，请关闭并重新打开订单后再检查。

## 何时需要找经理／管理员

- `仅限经理`：如果出现经理授权提示而你无法继续。
- 折扣选项与门店政策不一致。
- 折扣已应用，但结账被付款规则警告阻止。
