title: 术语
---

| 术语 / 标签 | 含义 |
| --- | --- |
| `Ordering` | 用于桌台/订单操作的主销售模块 |
| `Table` | 进入订单前的楼面/桌台选择页面 |
| `Order` | 包含购物车和操作项的交易详情页面 |
| `Checkout` | 用于金额、分单、订金或按品项分拆的付款面板 |
| `Void Txn` | 作废进行中的交易 |
| `Return` | 对已完成交易创建负数交易 |
| `Reopen` | 重新打开已结账/已完成交易以继续处理 |
| `Pay In / Pay Out` | 现金抽屉收支记录操作 |
| `Day-end` | 来自 `Manager Tools` 的营业日结算流程 |
| `POS Code` | 多数设备/配置操作所需的终端代码 |
| `Item Mapping` (`Main` / `Alt`) | 各语言显示的品项名称来源设置 |
| `Login Quick Switch` | 登录页面上的语言按钮，用于快速切换语言 |
