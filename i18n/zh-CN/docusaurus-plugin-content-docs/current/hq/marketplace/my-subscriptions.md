title: 我的订阅
---

:::info[适用对象]
需要检查 X1 POS 设备席位，或管理当前品牌启用中市集订阅的管理员。
:::

## 开始前

- 在左上角选择器确认正确的工作区和品牌。
- 只有在当前平板已损坏、遗失或永久替换时，才释放 POS 设备席位。
- 释放旧设备前，先确认替换平板已准备好注册。
- 先确认移除市集订阅是否会影响营业中的门店或集成。
- 如果已从某台终端删除 POS 门店配置，请在本页确认相关席位已重新变为可用。

## 打开此页面

打开 `我的订阅`。

此页有两个分页：

- `X1 产品`：检查 X1 POS 设备席位，并从旧平板释放席位。
- `市集`：检查和管理市集产品订阅。

## 本页控制内容

`我的订阅` 控制两类订阅。

### X1 POS 设备席位

使用 `X1 产品` 检查所选品牌的 POS 订阅席位。

`X1 POS 设备席位` 表格会显示：

| 字段 | 说明 |
| --- | --- |
| `门店` | 席位所属门店。 |
| `状态` | `已占用` 表示已有设备使用席位。`可用` 表示替换平板可注册到此席位。 |
| `设备` | 可用时显示 POS 设备名称。如果未提供设备名称，HQ 会显示缩短的设备 ID 和 `未提供设备名称`。 |
| `启用时间` | 设备占用席位的时间。 |
| `最后上线` | 该 POS 设备最后一次检查订阅的时间。 |
| `订阅期间` | 如有设置开始或结束日期，会显示席位期间。 |
| `操作` | 已占用席位的 `释放设备` 操作。 |

表格上方摘要会显示 `总席位`、`已使用席位` 和 `可用席位`。

### 市集订阅

使用 `市集` 检查当前品牌已订阅的市集产品。

根据产品不同，页面可能会提供以下操作：

- `设置`
- `取消订阅`

`设置` 只会出现在有直接配置页面的产品。由市集管理的 `3rd Party Platforms` 通常要改到 [General Settings](../online-ordering/general-settings.md) 和 [3rd Party Platforms](../online-ordering/third-party-platforms.md) 继续处理。

## 步骤

### 释放已替换的 POS 平板

1. 打开 `我的订阅`。
2. 选择 `X1 产品`。
3. 检查 `总席位`、`已使用席位` 和 `可用席位`。
4. 找到受影响门店和设备的行。
5. 检查 `设备`、`启用时间` 和 `最后上线`，确认这是旧平板。
6. 在 `操作` 选择 `释放设备` 图标。
7. 阅读警告信息。
8. 确认释放。

确认后，该行会变为可用，替换平板可以注册到释放出的席位。

### 管理市集订阅

1. 打开 `我的订阅`。
2. 选择 `市集`。
3. 找到要管理的订阅。
4. 如果产品有设置页面，选择 `设置`。
5. 如果产品已不再需要，选择 `取消订阅`。
6. 如系统有提示，请确认取消订阅。

## 保存后的影响

释放 POS 设备席位不会取消订阅。它只会清除旧设备占用，让另一部平板可以注册。旧平板再次上线并检查订阅时，会失去访问权。

POS 终端启用或验证订阅时会记录设备名称。较早已占用的席位，在该平板使用当前 app 回报前，可能会显示缩短设备 ID 和 `未提供设备名称`。

当 POS 终端删除门店配置时，app 会先释放该终端占用的 POS 订阅席位，再删除本机门店配置。如果 POS 显示 `无法停用订阅`，请先让管理员或支持在 HQ 确认席位状态，不要继续删除本机设置。

移除市集订阅后，相关功能或集成可能会立即不可用。

对于由市集管理的线上点餐平台，取消订阅也可能移除相关渠道可用性，并清除门店级已保存的平台设置。请将 `取消订阅` 视为高影响变更。

## 如何检查你的变更

1. 如已释放 POS 平板，确认席位由 `已占用` 变为 `可用`。
2. 注册替换平板，确认席位重新变为 `已占用`。
3. 确认替换平板可通过 POS 订阅检查。
4. 如保留市集订阅，确认产品仍出现在列表中。
5. 如移除市集订阅，确认它已从启用中订阅列表消失。
6. 如果您移除了平台订阅，营业前请重新检查 [Channel Settings](../online-ordering/channel-mapping.md) 或 [3rd Party Platforms](../online-ordering/third-party-platforms.md)。

## 如发生问题

- 如果显示 `请选择品牌以查看 X1 POS 订阅席位。`，请先选择正确品牌。
- 如果显示 `此品牌找不到 X1 POS 订阅席位。`，请确认该品牌已分配 X1 POS 席位。
- 如果显示 `X1 POS subscription database is unavailable`，请联系支持检查 production 订阅数据库连接。
- 如果 `释放设备` 无法使用，代表席位已可用，没有设备被指派。
- 如果释放了错误设备，请重新注册正确平板，或在旧平板被再次使用前联系支持。
- 如果已删除 POS 门店配置，但席位仍显示 `已占用`，请联系支持检查该终端释放席位时是否失败。
- 如果是误取消订阅，请回到 [市集](./catalog.md) 重新订阅（如产品仍可使用）。

## 何时请店主／管理员协助

- 您不确定应释放哪一部平板。
- 该平板可能仍在营业中的门店使用。
- 订阅会影响营业中的支付、点餐或财务流程。
- 您不确定是否仍有其他团队或门店在使用该市集产品。
