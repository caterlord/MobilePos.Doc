# 行动呼吁（Upselling）

![行动呼吁 设置](/img/hq/online-ordering/oo_call_to_action_page.png)

**行动呼吁（CTA）** 模组是一种很强的营收工具。它可以直接在顾客的手机网页或原生应用结帐流程里插入促销横幅，或加购提示。

## 放置位置

您可以依据顾客目前所在的位置，设置不同的 CTA 行为：

1. **购物篮位置**：顾客点选购物篮准备结帐时出现。适合放 `"加一杯饮料只要 $2"` 或 `"距离免运只差 $5"` 这类提示。
2. **订单历史位置**：付款后在电子收据页面出现。适合放会员注册连结，或 `"评价您的餐点"` 这类提示。

## 设置详情

要启用某个 CTA 区块，请按对应卡片打开编辑器。

- **标题与说明**：横幅的主标题和副文字，例如 `口渴了吗？`、`加一杯冰茶，让这餐更完整。`
- **目标智能分类**：这是实际触发条件。如果您在这里选择 `Drinks` 分类，顾客点选 CTA 横幅时，系统会立刻弹出一个视窗，显示 `Drinks` 分类里的所有商品，让顾客可以直接加进购物篮，不必回到主菜单。
- **动作网址**：如果您想直接把顾客导向别的页面，例如调查网站，请在这里输入完整的 `https://` 连结，而不是智能分类。

:::warning[翻译文字]
如果门店有双语运营，请记得填写 **Alt Title** 和 **Alt Description**。否则使用第二语言的顾客会看到空白。
:::
