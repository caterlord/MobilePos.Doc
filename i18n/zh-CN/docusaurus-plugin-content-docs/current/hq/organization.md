# 组织管理

:::info[适用对象]
品牌业主和总经理，需要邀请新的后台员工使用 HQ，或设置公司层级资料。
:::

## 开始前

- 确认您位于正确的组织。
- 先决定这次是要更改业务资料、HQ 成员存取，还是账单资料。

## 打开此页面

打开 `Organization Management`。

## 此页控制内容

`Organization Management` 是 HQ 的公司层级区域。一个组织可以包含多间门店或分店。

![组织管理](/img/manual/en/hq/organization.png)

## 主要分区

### 一般设置

在这里更新运营的高层资料，包括公司正式名称和商业注册资料。

### 成员与角色

您可以在这里邀请新的团队成员存取 HQ 后台。

:::note[POS 用户与 HQ 成员]
在这里邀请成员，代表他们可以使用 **HQ 网页后台**。
如果您只是要为新收银员建立一个能在店内终端机使用的登录账号，请改到 **POS 设置 -> POS Users**。
:::

### 邀请员工
1. 按一下 `Invite member`。
2. 输入新经理或办公室员工的电邮地址。
3. 选择他们的角色，例如 Admin 或 Viewer。
4. 对方会收到一封可登录 HQ 的邀请电邮。

### 账单

如果您直接负责账单，可以在这里管理付款方式，并查看 X1 软件订阅的过去发票。

## 保存后的影响

这里的更改会影响 HQ 管理，不会影响 POS 上的收银员登录。

## 如何检查你的更改

1. 保存后确认更新后的业务资料已经显示。
2. 如果您邀请了成员，请确认新成员纪录或邀请已经出现在清单里。
3. 如果更改和订阅或账单有关，也请一并查看相关的 [市集](./marketplace/catalog.md) 页面。

## 如果出现问题

- 确认您编辑的是 HQ 成员存取，不是 [POS Users](./pos-settings/pos-users.md)。
- 检查问题是出在组织设置，还是市集订阅管理。

## 何时需要找老板／管理员

- 更改会影响组织拥有权、账单，或其他管理员的存取。
- 您不确定对方应该是 HQ 成员还是 POS 用户。
