---
title: 更多工具
---

:::info[适用对象]
负责运营控制的经理和授权员工。
:::

## 开始前

- 你已使用所需角色登录
- 门店制度允许在这台终端上执行经理工具操作
- 你的 POS 用户群组包含该工具权限，或现场有经理可授权

## 这个页面的用途

`更多工具` 是经理操作中心。

:::note[范围说明]
`更多工具` 用于经理级运营控制（`日结`、`代收 / 代支`、钱箱/队列工具）。
硬件设备设置和打印机排查请使用 `设备`。
:::

## 可用工具

- `开钱箱`
- `日结`
- `代收 / 代支`
- `重置桌台计数`（仅快速点餐模式）
- `取餐控制台`（如已启用）
- `排队控制台`（如已启用）

## 步骤

1. 点击 `更多工具`。
2. 选择需要的工具卡片。
3. 完成该工具的确认流程。
4. 如果 POS 要求 `输入授权员工编号`，请已授权经理或员工输入自己的代码。

## 出现问题时

:::warning[工具卡片缺失]
某些工具取决于角色、平台或配置。
:::

:::warning[`权限不足`]
你的 POS 用户群组不允许该操作。请经理/管理员在 HQ `POS Users` 检查你的群组。
:::

## 何时找经理/管理员

- 你没有该工具的权限
- 快速点餐 / 取餐 / 排队控制与门店设置不一致
