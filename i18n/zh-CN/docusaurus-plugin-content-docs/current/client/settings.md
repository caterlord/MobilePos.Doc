## 开始前

- 你已登录并拥有设置权限
- 你了解门店对配置变更的制度

## 这个页面的用途

使用 `设置` 来维护终端行为、服务模式、语言 / POS 代码、打印、整合和运营规则。

## 语言配置

在 `常规` 中的语言区块里，你可以管理：

- 这台终端的默认界面语言
- 各语言的项目名称显示来源（`主语言` 或 `备用语言`）
- 登录画面中各语言是否显示快捷切换

:::tip[如何理解 `主语言` / `备用语言`]
`主语言` 和 `备用语言` 是商品设置里的两组项目名称来源。  
如果某个项目在其中一个来源里是空白，POS 会自动改用另一个可用名称。
:::

## 主要区块

<Tabs>
  <TabItem value="general" label="常规">
包含语言配置（`项目映射`、`登录快捷切换`）和必填代码（`收银机代码`、`收银柜代码`）。
  </TabItem>
  <TabItem value="operation" label="操作">
包含 `终端` 模式，以及订单 / 付款行为切换和服务流程控制。
  </TabItem>
  <TabItem value="system" label="系统">
包含 `数字` 里的控制项，例如 `四舍五入方式`、`小数位数` 和 `仅现金四舍五入`。
  </TabItem>
  <TabItem value="printing" label="打印">

  </TabItem>
  <TabItem value="platforms" label="第三方平台 + ODO">

  </TabItem>
  <TabItem value="report" label="报表">
包含与报表输出相关的设置，例如 `打印数量` 和 `日报表打印部分`。
  </TabItem>
</Tabs>

## 常见语言任务

<Tabs>
  <TabItem value="set-ui-language" label="设置默认界面语言">
1. 点击 `设置` -> `常规`。
2. 在语言配置中，选择你要设为默认的语言行。
3. 确认界面文字已经切换到目标语言。
4. 点击 `保存`。
  </TabItem>
  <TabItem value="item-mapping" label="设置项目名称映射">
1. 点击 `设置` -> `常规`。
2. 在目标语言行中，把 `项目映射` 设为 `主语言` 或 `备用语言`。
3. 点击 `保存`。
4. 打开 `点餐`，确认项目名称显示正确。
  </TabItem>
  <TabItem value="quick-switch" label="配置登录快捷切换">
1. 点击 `设置` -> `常规`。
2. 为员工需要在登录画面切换的语言开启 `登录快捷切换`。
3. 点击 `保存`。
4. 返回登录画面，确认语言快捷切换按钮已经出现。
  </TabItem>
</Tabs>

## 切换终端模式

<Tabs>
  <TabItem value="restaurant" label="Restaurant">
1. 点击 `设置` -> `操作`。
2. 在 `终端` 中，把 `模式` 设为 `Restaurant`。
3. 点击 `设置` -> `常规`。
4. 将 `启用堂食自动开单` 关闭。
5. 将 `启用外卖自动开单` 关闭。
6. 如果门店使用楼层平面图视图，开启 `启用楼层平面图模式`。
7. 点击 `保存`。
8. 返回 `点餐`，确认员工在加项目前必须先选择餐台或外卖订单。
  </TabItem>
  <TabItem value="quick-order" label="Quick Order">
1. 点击 `设置` -> `操作`。
2. 在 `终端` 中，把 `模式` 设为 `Quick Order`。
3. 点击 `设置` -> `常规`。
4. 将 `启用堂食自动开单` 关闭。
5. 将 `启用外卖自动开单` 打开。
6. 如果外卖应该先打开，就开启 `启用默认外卖模式`；如果堂食应该先打开，就保持关闭。
7. 如果你希望 POS 为新快速点餐自动轮转台号，就将 `启用堂食自动分配台号` 和 `启用外卖自动分配台号` 打开。
8. 点击 `保存`。
9. 返回 `点餐`，确认 POS 会直接打开新订单，而不是先让员工选餐台。
  </TabItem>
</Tabs>

:::note[相关设置]
只有在对应的自动开单设置已开启后，`启用堂食自动分配台号` 和 `启用外卖自动分配台号` 才会显示。
:::

:::tip[快速点餐工具]
`更多工具` 里的 `重置桌台计数` 会在门店使用轮转快速点餐计数时配合快速点餐模式使用。
:::

## 常见四舍五入任务

1. 点击 `设置` -> `系统`。
2. 在 `数字` 中选择 `四舍五入方式`（`无`、`无条件进位`、`无条件舍去` 或 `四舍五入至最接近5分`）。
3. 在显示该字段时设置 `小数位数`。
4. 当四舍五入只应作用于现金付款时，打开 `仅现金四舍五入`。
5. 点击 `保存`。
6. 打开 `结账`，确认现金和非现金方式的总额表现符合预期。

如果 `四舍五入方式` 设为 `四舍五入至最接近5分`，`小数位数` 会隐藏。

## 从备份还原设置

1. 点击 `设置`。
2. 在底部操作栏点击 `还原`。
3. 选择你要应用的备份项。
4. 确认还原完成。
5. 检查关键设置，然后点击 `保存`。

目前版本的还原会覆盖更多配置区域，所以还原后请检查关键项目：

- 打印机路由和打印设置
- 顾客显示和终端行为设置
- 整合相关设置

## 常见打印语言覆盖任务

1. 点击 `设置` -> `打印`。
2. 打开 `模板`。
3. 选择目标打印模板。
4. 将 `语言` 设为需要的打印语言（或保持 `跟随系统`）。
5. 点击 `保存`。
6. 打印测试收据 / 单据，并确认输出语言正确。

## 常见收据自动打印任务

1. 点击 `设置` -> `打印`。
2. 打开 `设置`。
3. 在 `收据` 中，如果收据应在结账成功后自动打印，就打开 `自动打印收据`；如果要让员工手动决定，就关闭它。
4. 点击 `保存`。
5. 完成一次正常结账，并确认收据打印符合门店制度。

## 常见外卖联络资料打印任务

1. 点击 `设置` -> `打印`。
2. 打开 `设置`。
3. 在 `收据` 中，打开或关闭 `在收据上显示订单联络资料`。
4. 在 `厨房单` 中，打开或关闭 `在厨房输出上显示订单联络资料`。
5. 点击 `保存`。
6. 打印外卖收据或厨房单，并确认联络资料只会显示在门店需要的位置。

:::note
如果你需要调整这一部分的收据字号，请打开 `模板`，并查看收据字体设置中的 `Customer Contact` 行。
:::

## 常见日结报表区块任务

1. 点击 `设置` -> `报表`。
2. 如果门店需要多份纸本，设置 `打印数量`。
3. 在 `日报表打印部分` 中，按经理在收店时需要打印的内容，打开或关闭 `摘要`、`现金流动`、`销售明细` 和 `付款明细`。
4. 如果你需要控制这些区块里的细项，请打开 `显示进阶部分`。
5. 在 `进阶部分` 中，打开或关闭 `开机金额`、`代收`、`代支`、`现金差额`、`结柜金额`、`稽核`、`收银员摘要`、`部门销售` 或 `付款摘要` 等项目。
6. 点击 `保存`。
7. 运行一次日结报表，并确认只会打印需要的区块。

:::note
只有在对应的上层区块保持开启时，进阶区块开关才会生效。
:::

## 整合与渠道显示行为

- 在 `第三方平台` 中，只会显示已启用的整合。
- 在 `ODO` -> `渠道映射` 中，只会显示已映射的渠道。
- 如果终端离线，渠道映射会显示仅在线可用的提示，而且映射行会变为不可用。

## 步骤

1. 点击 `设置`。
2. 在左侧面板选择对应区块。
3. 更新所需字段 / 开关。
4. 点击 `保存`。
5. 确认更改已经生效。

- `报表` 包含 `打印数量`、`日报表打印部分` 和 `显示进阶部分`
- `设置` 底部的 `还原` 操作

## 出现问题时

:::warning[无法保存设置]
检查必填字段（`收银机代码`、必填代码）后重试。
:::

## 何时找经理/管理员

- 你没有权限更改设置
- 更改可能影响所有终端或整个门店的行为
