---
title: 配置输入设备
---

:::info[适用对象]
配置扫描器或输入硬件的经理或授权员工。
:::

## 开始前

- 你已经使用设备设定权限登录
- `POS Code` 已设定

## 步骤

1. 打开 `Devices`。
2. 打开 `Input Devices` 页签。
3. 需要时点按 `Info` 检查状态。
4. 点按目标设备的编辑图示。
5. 更新所需栏位，然后点按 `Save`。

## 你应该会看到

- 输入设备列表与状态指示
- 每一列都有 `Info` 与编辑动作

> 待补截图：`client/cl-35-devices-input-devices-tab-list.png`

如果是条码／QR 扫描器，编辑对话框可能会包含：

- `Connection`
- `IP Address` / `Port`（IP 模式时）
- `Scanner Preview`
- `Camera`（镜头方向选择）

> 待补截图：`functions/fn-input-barcode-scanner-dialog.png`

## 如果出现问题

:::warning[扫描器无法读取]
先确认所选连接模式与相机／主机网络设定，然后再重试。
:::

## 什么时候要找经理／管理员

- 输入设备在保存后仍不可用
- 需要的扫描器选项不见了
