---
title: 配置输入设备
---

:::info[适用对象]
配置扫描器或输入硬件的经理或授权员工。
:::

## 开始前

- 你已经使用设备设定权限登录
- `POS Code` 已设定

## 步骤

1. 打开 `Devices`。
2. 打开 `Input Devices` 页签。
3. 需要时点按 `Info` 检查状态。
4. 点按目标设备的编辑图示。
5. 更新所需栏位，然后点按 `Save`。

## 你应该会看到

- 输入设备列表与状态指示
- 每一列都有 `Info` 与编辑动作

> 待补截图：`client/cl-35-devices-input-devices-tab-list.png`

如果是条码／QR 扫描器，编辑对话框可能会包含：

- `扫描输入来源`
- `相机`
- `USB / 蓝牙`
- `扫描预览`
- `相机`（镜头方向选择）
- `测试 USB/蓝牙扫描器`
- `启用相机备用`

> 待补截图：`functions/fn-input-barcode-scanner-dialog.png`

使用 USB 或蓝牙扫描器时，选择 `USB / 蓝牙` 作为默认扫描方式。先在 `测试 USB/蓝牙扫描器` 中扫描，确认扫描值出现后再点按 `Save`。

如果希望 POS 优先打开相机，请选择 `相机`。如果默认使用 USB/蓝牙，但员工可能需要用相机备用，请打开 `启用相机备用`。

## 如果出现问题

:::warning[扫描器无法读取]
先确认所选扫描输入来源。USB/蓝牙扫描器需要能像键盘一样输入文字并发送 Enter。相机扫描请检查相机权限和镜头方向设定，然后再重试。
:::

## 什么时候要找经理／管理员

- 输入设备在保存后仍不可用
- 需要的扫描器选项不见了
