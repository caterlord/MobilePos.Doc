---
title: 更换桌台
---

:::info[适用对象]
需要把进行中的订单移到其他桌台的收银员与班次经理。
:::

## 开始前

- 你已经登录并在订单画面
- 你已经打开正确订单
- 你可以打开功能菜单（三点）
- 你已经送出尚未发送的品项编辑
- 你知道目标是空桌台

## 步骤

1. 点按功能菜单（三点）。
2. 在 `Select Function` 点按 `Change Table`。
3. 如果 `Change Table` 在另一页，点按 `Previous` 或 `Next`。
4. 如果你需要切换桌台列表与楼面图，点按 `Change Table` 右下角的切换按钮。
5. 如果你需要分桌，点按 `Show Split`。
6. 点按目标桌台。
7. 等待桌台选择器关闭。
8. 如果应用回到桌台列表，请从目标桌台重新打开已移动的订单。

## 你应该会看到

- 订单会从原桌台移走
- 订单会出现在目标桌台
- 你可以在目标桌台继续点餐
- 右下角切换按钮会在 `Table List` 与 `Floor Plan` 之间切换（如果楼面图模式已启用）
- 显示分桌后，`Show Split` 会变成 `Hide Split`

## 如果出现问题

- 如果看到 `No item to change table`，请先加至少一个品项。
- 如果看到 `Cannot change table` 与 `Please save all operations before changing table`，先点按 `Send` 再重试。
- 如果在 `Floor Plan` 找不到目标桌台，请切换到 `Table List`。
- 如果目标分桌不见了，请点按 `Show Split`。
- 如果看到 `Please select an empty table`，请改选其他桌台。
- 如果看到 `Unable to change table`，请在刷新订单后只重试一次。

## 什么时候要找经理／管理员

- `Manager only`：移动涉及特殊处理或争议案例的订单。收银员的替代做法是保留当前桌台并继续服务。
- 如果没有可用的目标桌台，请找经理。
- 如果重试一次后仍失败，请找管理员。
