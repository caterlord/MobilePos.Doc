:::info[适用对象]
所有前线员工。
:::

## 开始前

- 终端已在线
- 你知道自己的 PIN

## 步骤

1. 如果需要更改登录语言，可以点按登录画面的语言快速切换按钮（`EN` / `繁` / `简`）。
2. 在登录画面输入 PIN。
3. 确认 PIN 输入正确。
4. 点按 `SIGN IN`。

## 你应该会看到

- 成功登录后会进入 `Ordering` 页面。

![Login PIN screen](/img/manual/en/getting-started/gs-06-login-passcode-screen.png)

## 如果出现问题

:::warning[登录失败]
先检查 PIN，然后只重试一次。
:::

## 什么时候要找经理／管理员

- 重试后 PIN 还是失败
- 订阅或离线提示阻止你登录
