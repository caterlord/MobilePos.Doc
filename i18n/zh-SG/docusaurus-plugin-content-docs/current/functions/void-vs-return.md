import Tabs from '@theme/Tabs';

:::info[规则]
进行中的交易用 `Void`。已完成的交易用 `Return`。
:::

## 步骤

<Tabs>
  <TabItem value="void" label="Void">
1. 打开进行中的订单。
2. 点按功能菜单。
3. 在 `Select Function` 点按 `Void Txn`。
4. 如果 `Void Txn` 在另一页，点按 `Previous` 或 `Next`。
5. 确认提示。
  </TabItem>
  <TabItem value="return" label="Return">
1. 打开已完成交易。
2. 点按 `Return`。
3. 确认提示。
4. 完成退款付款流程。
  </TabItem>
</Tabs>

:::

- 高金额退款／作废
- 第三方连结付款的限制
