# Model Context Protocol (MCP)

使用 X1 的 MCP 服务器，让 AI 代理可以与 X1 API 交互。

X1 的 Model Context Protocol (MCP) 服务器提供一组工具，让 AI 代理可以安全地查询 X1 API、手册页面与支援文章。

## 如何开始

1. 如果你要连接 Cursor、Claude Desktop 或 ChatGPT，请按照英文页面中的最新配置步骤进行。
2. 如果要确认支持的工具列表，请参考下方的 MCP 工具说明。
3. 如果你要回报 MCP 相关问题，请使用 `mcp@x1.tech`。

:::info[语言说明]
MCP 配置文件、OpenAPI URL 与工具名称会保留原文，以免影响实际接入设置。
:::
