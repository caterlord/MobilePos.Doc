# 分类与商品

![Online Categories](/img/hq/online-ordering/oo_categories_page.png)

**分类** 页面是您把内部 POS 商品公开到线上通路的主要工具。默认情况下，在菜单目录里建立的商品不会出现在网上点餐，直到您在这里明确对映它们。

## 1. 发布分类
与其一笔一笔发布商品，不如直接把整个 **智能分类** 或 **标准分类** 一起对映。
完成对映后，分类里的任何菜单商品都会变成可线上购买。

## 2. 图片管理
顾客很看重视觉。这个模组就是您上传商品照片的地方。
- 点选商品或分类列，打开编辑器。
- 上传图片后，系统会自动压缩并托管到 CDN。

## 3. 线上定价策略
店内卖 $10 的汉堡，在外送应用上可能要卖 $12，才足以抵销平台佣金。
在这里商品的价格分页中，您可以设定 **Online Base Price Override**。这样线上价格就会和终端 POS 价格完全分开。

## 4. 商品说明
使用 **Description** 栏位加入营销文案或过敏原提醒。这段文字会显示在数码菜单里的商品名称下方。

:::tip[临时售罄处理]
如果某个商品暂时缺货，**不要**把它从线上分类里删除。请改用 POS 终端的 **Inventory** 应用，把商品标记为 "Sold Out"。这样它会立即在顾客手机上变灰，但不会破坏您的菜单结构。
:::
