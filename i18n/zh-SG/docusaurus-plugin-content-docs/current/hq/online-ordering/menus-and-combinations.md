# 线上菜单

![Online Menus 设定](/img/hq/online-ordering/oo_menus_page.png)

**线上菜单** 模组决定已发布的分类会如何组合成可浏览的数码目录。

它的运作方式和 POS 菜单类似，但只限于网页和 App 使用者。

## 菜单组合

“菜单组合”就是外层包装，例如 `Lunch Delivery Menu` 或 `Dinner Pickup Menu`。

顾客打开您的店铺连结时，浏览的就是某一个特定菜单组合。

### 建立菜单
1. 点选 **New Menu** 建立外壳。
2. 定义 **Code**（供 API 连结使用）和 **Display Name**（网页应用页面标题会显示的名称）。
3. 选择哪些 **Categories**（就是您在上一步设定的分类）要映射到这个菜单里。
4. 拖放分类，整理它们在顾客手机画面上的显示顺序。

### 指派店铺
线上菜单是全域性的，但一定要明确映射到实体店铺。
如果 `Shop A` 和 `Shop B` 的商品完全一样，您只需要建立一次菜单，再指派给两间店铺即可。
但如果 `Shop A` 有卖酒，`Shop B` 没有，您就应该建立两个不同的线上菜单，并分别对映。
