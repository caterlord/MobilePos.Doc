---
title: 按钮样式
---

:::info[适用对象]
负责管理分类或商品显示所用的可重复按钮外观设定的管理员。
:::

## 开始前

- 确认正确的品牌。
- 先决定您是需要新增样式，还是可以重用现有样式。

## 打开此页面

打开 `Menu Management` -> `Button Styles`。

## 本页用途

`Button Styles` 用来存放可重复使用的视觉样式，之后可以套用到菜单设定中的其他按钮。

您可以在这里：

- 新增按钮样式
- 检视现有样式
- 更新非系统样式
- 删除不再需要的非系统样式

## 主要分区

### 样式表格

主表格会显示：

- `ID`
- `Style Name`
- `Preview`
- `Color Code`

带有 `System` 标记的样式是系统样式，不能编辑或删除。

### 样式编辑器

编辑器会包含以下栏位：

- `Style Name`
- `Background Color`

常见动作包括：

- `Add Button Style`
- `Create`
- `Update`
- `Delete`

## 步骤

1. 打开 `Menu Management` -> `Button Styles`。
2. 按 `Add Button Style` 建立新样式，或打开现有样式编辑。
3. 输入 `Style Name`。
4. 选择 `Background Color`。
5. 检查预览。
6. 保存样式。

## 保存后的影响

保存样式会更新可重复使用的预设，但不会自动证明所有相关分类或商品都已经正确套用，所以请再确认样式实际套用到哪里。

## 如何检查你的更改

1. 回到样式表格，确认该样式已经列出。
2. 检查预览和颜色。
3. 打开使用该样式的分类或其他设定，确认选择的是正确样式。

## 如果出现问题

- 确认您不是在编辑 `System` 样式。
- 检查样式是否已经保存，但还没有指派到分类或商品。
- 如果该样式已经不需要，请先确认它没有在使用中，再删除。

## 何时需要找老板／管理员

- 样式更改会影响多组现场菜单，而且您无法全部确认。
- 您不确定共用样式是否被其他店铺或团队使用。
